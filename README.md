## Overview

Over the course of this Summer of Code project, I focused on building strong, pattern-based problem-solving skills by covering all major data structures and algorithms required for coding interviews and competitive programming. The learning process combined theoretical understanding — using Striver's A2Z DSA Course — with extensive practice on mentor-assigned problem sheets released every two days, each containing 9–10 curated questions per topic. This ensured that every concept was reinforced through immediate application rather than just learning.

By the end of the project, I have covered the complete topics — starting from arrays and basic traversal techniques, all the way through dynamic programming and graph algorithms. Alongside this, I maintained a personal set of notes in VS Code, with a separate file created for every topic, serving as a structured revision resource going forward.
